# stylus 语法之 运算符

## 简介

下面就是运算符，优先级从高到低：

```
[]
 ! ~ + -
 is defined
 ** * / %       // ** 表示指数
 + -
 ... ..         // ... 是开区间， .. 闭区间
 <= >= < >
 in
 == is != is not isnt
 is a
 && and || or
 ?:
 = := ?= += -= *= /= %=
 not
 if unless
 ```

 大体上和常见的运算符意义一样，不同的有：

 `**` 表示指数运算：

 ```
 2 ** 8
// => 256
```

*******

`...` 和 `..` 表示一个范围(range)，比如：

```
1..5 // =>  1 2 3 4 5
1...5 // => 1 2 3 5
```

## 颜色运算

#### rgb 的加减

简单的对位相加，如果和超过 `f` ，那么就去 `f` 。
相减时如果差为负值，则取 `0` 。

```
#000+#010  //=>#010
#010+#010  //=>#020
#0e0+#020  //=>#0f0
```

#### 按百分比增减亮度

这个应该能很容易搞一个很舒服的渐变……

```
#888 + 50%
// => #c3c3c3

#888 - 50%
// => #444
```

#### 色相的计算

```
#f00 + 50deg
 // => #ffd500
 ```
